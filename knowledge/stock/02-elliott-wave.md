# 波浪理论（Elliott Wave）

> 拉尔夫·纳尔逊·艾略特创立，通过股价走势的周期性规律预测未来。

---

## 核心概念

### 八浪循环

市场的基本形态是**五浪上涨 + 三浪下跌**：

```
上涨5浪：
1浪：起始上升（建仓）
2浪：回调（洗盘）
3浪：主升浪（最长，通常最长）
4浪：回调（整理）
5浪：冲顶（出货）

下跌3浪：
A浪：下跌（出逃）
B浪：反弹（逃命）
C浪：主跌浪（最狠）
```

### 波浪的铁律

1. **2浪不能跌破1浪起点**（否则不是推动浪）
2. **3浪永远不是最短的一浪**（通常是最长）
3. **4浪不能与1浪重叠**（除非是斜线三角形）
4. **交替原则**：2浪和4浪形态交替（一个深回调，一个浅回调）

## 为什么要学波浪

量化大量使用波浪理论来：
- 识别当前处于哪个浪（3浪启动？5浪冲顶？）
- 预测3浪和5浪的目标位
- 在B浪反弹时出货

**我的机会**：
- 识别量化是否在"数浪"
- 在关键浪的起点/终点反向操作
- C浪是最狠的下跌，不能接飞刀

## 波浪的局限性

1. **主观性**：不同人数出不同的浪
2. **事后验证**：事后看很清楚，当时模糊
3. **量化不用**：量化主要用趋势跟踪，不主动数浪
4. **不精确**：无法给出精确的买卖点

**实用价值**：
- 作为辅助工具，配合道氏理论使用
- 帮助判断"大概在什么位置"（顶部？底部？中部？）
- 配合RSI、MACD确认

---

## 综合使用

波浪 + 道氏 = 更完整的判断

- 道氏告诉我：趋势向上/向下
- 波浪告诉我：现在大概在哪一段，可能还有多少空间
