## 核心理论

### 一、分型

股价走势最基本的结构单元：

- **顶分型**：三根K线，中间一根最高
- **底分型**：三根K线，中间一根最低

```
顶分型：  ┌─
          │  中间最高
        ─┘

底分型：  └─
          │  中间最低
        ─┐
```

**意义**：分型是趋势反转的最小单元

### 二、笔

两个相邻的顶底分型之间构成一笔：

- 上升笔：底分型 → 顶分型
- 下降笔：顶分型 → 底分型

**要求**：至少有5根不含包含关系的K线

**意义**：笔是趋势的基本构成

### 三、线段

三个以上的重叠部分构成线段：

```
重叠区域 ← 这是中枢的雏形
```

**意义**：线段是更稳定的趋势结构

### 四、中枢（核心概念）

**中枢是股价波动的核心区域**

- 三个连续的次级别走势的重叠部分
- 中枢之上看多，中枢之下看空
- 中枢的扩展和新生决定趋势

```
    上轨
    ┌─────┐
    │中枢 │ ← 三段重叠
    └─────┘
    下轨
```

### 五、背驰（关键信号）

**背驰 = 价格创新高/低，但力度减弱**

- 上升趋势：价格创新高，但MACD面积变小 → 顶背驰 → 卖出
- 下降趋势：价格创新低，但MACD面积变小 → 底背驰 → 买入

**意义**：背驰是最可靠的转折信号

### 六、三类买卖点

| 买卖点 | 位置 | 可靠性 |
|--------|------|--------|
| 第一类 | 中枢背驰 | 最高 |
| 第二类 | 中枢笔回抽 | 中等 |
| 第三类 | 中枢新生后的次级别回抽 | 较高 |
