# 5日均量上穿60日均量策略 v1.0

## 核心发现
**5日均量从下往上接近/上穿60日均量时买入 = 高胜率量化策略**

灵感来源：毅哥实战经验 + 数据回测验证

---

## 策略一：上穿前一天买入（主策略）

### 买入条件
1. **5日均量正在从下往上接近60日均量**
   - 量比 = 5日均量 / 60日均量 = 0.85~0.98
   - 明天可能上穿

2. **5日均量斜率 > 30**（量能明显加速）

3. **RSI < 52**（还没超买）

### 验证条件
- 底分型出现 = 加分
- 20日涨幅 < 20% = 加分

### 止损
- 如果5日均量重新掉到60日均量下方
- 止损 -5%

### 目标
- 上穿成功持有
- 回调到关键位加仓
- 等背驰或顶部信号卖出

## 策略三：洗盘识别与二次介入

### 主力洗盘信号
```
上穿成功后出现：
- 横盘不涨（3-5天）
- 缩量回调
- 回调到关键位（均线/中枢/前高）

= 主力洗盘，不是真跌
```

### 二次介入时机
- 洗盘结束特征：
  - 再次放量启动
  - 突破洗盘区间高点
  - 缩量到极致后放量

## 失败因子分析

### 导致上穿失败的特征
| 因子 | 表现 | 含义 |
|------|------|------|
| 缩量 | 量比 < 0.8 | 动力不足 |
| 无影线 | 上影 < 0.5% | 没有抛压被吸收 |
| 上影线 | 上影 > 2% | 上方有抛压 |
| RSI超买 | > 70 | 位置太高 |
| 前期大涨 | 20日涨幅 > 30% | 获利盘大 |

### 成功案例特征
| 特征 | 表现 | 含义 |
|------|------|------|
| 光头大阳 | 无上影线 | 主力控盘 |
| 下影线长 | > 2% | 下方支撑强 |
| 大跌后 | 次日平/涨 | 洗盘结束 |

## 完整交易流程

```
1. 扫描发现信号：
   - 5日均量线正在从下往上接近60日均量
   - 量比 0.85~0.98
   - 5日斜率 > 30
   - RSI < 52

2. 上穿前一天买入

3. 次日观察：
   - 涨幅 > 3% + 量增50%+ = 主力控盘 ✅ 持有
   - 涨幅 < 2% + 缩量 = 动力不足 ⚠️ 观察
   - 大跌 = 失败 ❌ 止损

4. 持有过程中：
   - 等出现顶部信号（背驰/顶部形态/巨量）
   - 或等洗盘信号

5. 洗盘时：
   - 不慌，等二次启动信号
   - 缩量到极致 + 再次放量 = 二次介入机会
```

## 期望值计算

```
成功：70% × +15% = +10.5%
失败：30% × -5% = -1.5%
期望值 = +9%每次
```

## 注意事项

1. 信号稀少，不要频繁交易
2. 止损要坚决，不要扛单
3. 洗盘时保持耐心
4. 结合大盘环境使用
