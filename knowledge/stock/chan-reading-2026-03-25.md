## 一、缠论数学基础（核心原理）

### 原理一：走势终完美 ⭐最核心
> 任何级别的任何走势类型终要完成

"走势终完美"是缠论的公理，是一切分析的基础。没有走势可以无限延伸而不结束。

### 原理二：中枢存在定理
> 任何级别任何完成的走势类型，必然包含一个以上的缠中说禅走势中枢

→ 没有中枢 = 没有趋势 = 不操作（纪律）

### 分解定理一
任何级别的任何走势，都能分解成同级别"盘整"、"下跌"与"上涨"三种走势类型的连接

### 分解定理二
任何级别的任何走势类型，都至少由三段以上次级别走势类型构成

## 二、中枢（核心结构）

### 定义
三个连续次级别走势类型重叠的部分 → ZG/ZD/GG/DD

### 中枢的三种演化
1. **延续**: 围绕中枢震荡，不离开
2. **扩张**: 离开后形成更大级别中枢
3. **新生**: 离开后形成新中枢（趋势产生）

### 中枢破坏
离开段不回到中枢 → 中枢破坏 → 趋势产生

### 中枢与走势必完美
"走势必完美" = 中枢必然完成 = 走势必然结束
→ 三个次级别是必须的，一定有两次回抽

---

## 三、背驰（动力学核心）

### 定义（趋势力度）
前一"吻"的结束与后一"吻"开始，由短线均线与长期均线相交所形成的**面积**

### 判断方法
1. **面积法**: 比较前后面积（稳妥但滞后）
2. **均线平均力度**: 面积÷时间（即时但风险大）
3. **MACD辅助**: 黄白线和柱子面积（最常用）

### 背驰-转折定理
某级别的趋势背驰将导致：
- 该趋势最后一个中枢的**级别扩展**
- 或该级别**更大级别的盘整**
- 或该级别**以上级别的反趋势**

### 小背驰-大转折定理
- **大级别向下**: 必要条件 = 该级别走势的最后一个次级别中枢出现**第三类卖点**
- **大级别向上**: 必要条件 = 该级别走势的最后一个次级别中枢出现**第三类买点**

### ABC段的力度比较
- C段不一定创新高
- 如果连新高都创不出，力度最弱
- 背驰/盘整背驰是走势分段的重要依据

## 四、三类买卖点

### 第一类买卖点（趋势转折点）
背驰产生的转折点。
- 买点：下跌趋势背驰的最低点
- 卖点：上涨趋势背驰的最高点

### 第二类买卖点
第一类之后的回调不创新低/高。
- 本质：一类买卖点之后的次级别回撤
- **买卖点定律一**: 第二类买卖点由次级别相应走势的第一类买卖点构成

### 第三类买卖点
离开中枢后回试不回到中枢。
- 买点：次级别向上离开，回试不跌破ZG
- 卖点：次级别向下离开，回试不升破ZD
- **意义**: 对付中枢结束，中枢扩张或新生的买点

### 三类买卖点的完备性定理 ⭐
> 市场必然产生赢利的买卖点，只有第一、二、三类

### 升跌完备性定理
> 市场中的任何向上与下跌，都必然从三类买卖点中的某一类开始以及结束

### 买卖点级别定理
> 大级别的买卖点必然是次级别以下某一级别的买卖点

## 五、分型、笔、线段（微观结构）

### K线包含关系处理（数学起点）

**方向定义：**
- **向上**: gn ≥ gn-1
- **向下**: dn ≤ dn-1

**包含关系:**
- 向上包含: gn > gn-1 且 dn < dn-1 → 去高留低
- 向下包含: gn < gn-1 且 dn > dn-1 → 去低留高

**重要**: 处理后的K线才能确定方向，才有分型

### 分型
经过包含处理后，三根K线构成:
- **顶分型**: 中间一根高点最高、低点也最高
- **底分型**: 中间一根低点最低、高点也最低

### 笔
顶分型和底分型之间的连接。
- 要求: 两个分型之间有不共用的独立K线

### 线段（特征序列定义）

**特征序列:**
- 向上线段的特征序列 = 所有向下笔(X序列)
- 向下线段的特征序列 = 所有向上笔(S序列)

**线段终结的两种情况:**

**第一种（无缺口）:**
特征序列第一和第二元素间无缺口 → 出现分型 → 线段结束

**第二种（有缺口）:**
特征序列第一和第二元素间有缺口 → 从分型高点/低点开始的笔的特征序列出现反向分型 → 线段结束

### 分型的当下判断
- 分型可修改 = 图形未完成
- 分型不可修改 = 图形必然完成
- 走势必完美 = 图形必然完成

## 七、均线轮动与板块强弱

### 板块轮动规律
- 一线 → 二线 → 三线节奏演绎
- 最后连三类都动起来 = 该板块大调整的标志
- 第二龙头补涨比第一龙头还有力 = 板块进入调整的标志

### 牛市节奏
1. 第一阶段：成分股（指数股）
2. 第二阶段：成长股
3. 第三阶段：重组股

### 均线辅助
- 5日线不破 = 强势
- 连5日线都碰不到 = 不可能形成日线中枢 = 极强

## 八、中阴阶段

### 定义
走势在两个明确方向之间的过渡状态（类似人死后的"中阴身"）

### 核心处理原则
- 1分钟走势 → 必须先有一个5分钟中枢（100%成立）
- 布林通道收口 = 对中阴结束时间的最好提示
- 中阴结束后必然面临中枢破坏（延伸或第三买卖点）

---

## 九、资金管理（第一生存法则）

### 核心原则
- **钱必须长期无压力**
- 投资是一生的游戏
- 大资金比拼的终极 = 资金管理水平

### 操作节奏
- 短线看30分钟买点（配合日线不在背驰段）
- 小资金用5分钟买点
- 中线看5周线，短线看5日线

### 仓位原则
- 100万以下资金不超过5只股票
- 有买点才买，没买卖点就等

## 十、缠师心法荟萃（实战精华）

### 买点原则
1. 只在第一、二、三类买点买
2. 所有买点都必然在调整时出现
3. 不追高，永远不追涨杀跌
4. 底部区间下探失败时买（最好买点）
5. 调整结束后启动时介入

### 卖点原则
1. 背驰段的区间套里陆续走
2. 卖点后的回抽如果不能回到原中枢，可能形成第三类买点
3. 不是等跌了才问卖不卖，而是涨时进入背驰段就走

### 操作节奏
1. 短线操作：看30分钟买点（最好日线不在背驰段）
2. 大级别买点介入，持有到大级别卖点（真正的长期投资）
3. 不参与中枢震荡，只搞最强势的（散户最该干的事）
4. 第三类买点持股，直到新中枢出现

### 止损原则
1. 破5日线（3天拉不回来 = 真跌破）
2. 有买点才买，没买卖点就等
3. 放量突破回调时买（风险小很多）

### 心态原则
1. 不追涨杀跌，没有股票值得追高买
2. 别把股票当慈善晚会，没信心的不下来，换手不充分不大涨
3. 思维转过来：太猛不是走的理由，该猛不猛或想猛猛不起来才是走的原因
4. 卖早卖错了，等短线买点再进

### 同级别分解原则
1. 不允许盘整里的中枢延伸，3段次级别就是了
2. 分解起始点必须是前面走势类型的结束点
3. 盘整可以很简单处理，按次级别一段段分解操作

## 十一、缠论经典定理汇总

| 定理 | 内容 |
|------|------|
| 完备性定理 | 市场必然产生赢利的买卖点只有第一、二、三类 |
| 升跌完备性 | 任何上涨下跌都必然从某类买卖点开始及结束 |
| 级别定理 | 大级别买卖点必然是次级别以下某级的买卖点 |
| 背驰-买卖点定理 | 任一背驰都必然制造某级别的买卖点 |
| 转折定理 | 任何级别上涨转折由某级别第一类卖点构成 |
| 背驰-转折定理 | 趋势背驰导致中枢级别扩展/更大级别盘整/反趋势 |
| 小背驰-大转折定理 | 小级别背驰引发大级别转折的必要条件是最后中枢出现三卖/三买 |
| 第一利润最大定理 | 确定操作级别下，操作模式利润率最大 |
| 第二利润最大定理 | 不同交易品种下，激进操作模式利润率最大 |
| 区间套定理 | 大级别转折点可通过不同级别背驰段逐级收缩确定 |
