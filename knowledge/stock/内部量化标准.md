# 内部量化标准 v1.0

## 四维体系
缠论（结构）+ 蜡烛图（时机）+ 量价（能量）+ 真技术（心法）

---

## 一、选股标准

### 入池条件（满足任意2条）
| 维度 | 标准 |
|------|------|
| 缠论 | 底分型出现 或 突破中枢ZG/ZD |
| 蜡烛图 | 锤子线/吞没/启明星 形态确认 |
| 量价 | RSI<40（超卖）或 量比>1.5（异动） |
| 真技术 | 平量/缩量止跌 或 大盘跌它不跌 |

### 出池条件
- RSI>80 超买3天不回落
- 放量跌破重要支撑
- 逻辑兑现后

---

## 二、买点标准（四维共振优先）

### 第一类买点（一买）
**同时满足：**
- 缠论：底背驰 或 趋势背驰
- 蜡烛图：锤子线 + 次日白实体确认
- 量价：缩量回调后平量止跌
- 真技术：RSI<30 超卖

### 第二类买点（二买）
**同时满足：**
- 缠论：一买后回调不破新低
- 蜡烛图：回调出现锤子线/吞没
- 量价：缩量回调
- 真技术：概率最大时

### 第三类买点（三买）
**同时满足：**
- 缠论：离开中枢后回试不回到ZD（向上）
- 蜡烛图：缩量回调不破关键位
- 量价：回调缩量
- 真技术：等确认后买入

### 仓位规则（真技术：机会→仓位）
| 信号强度 | 仓位 |
|---------|------|
| 单一信号 | 1-2成 |
| 2维共振 | 3-4成 |
| 3维共振 | 5成 |
| 4维共振 | 6-7成 |

## 三、止损标准

### 止损原则
> 止损是最低级的风控手段，治病于未发才是上策
> 买入前就设止损，不是买入后再想

### 止损位（按优先级）
1. **固定止损**：买入价 -5%
2. **结构止损**：跌破底分型低点
3. **中枢止损**：跌破最后一个中枢ZD
4. **逻辑止损**：买入逻辑不成立时

### 止损执行
- 破止损位立即走，不犹豫
- 不做"快刀慢割"——直接止损，不分批

## 四、持有标准

### 持有条件
| 维度 | 标准 |
|------|------|
| 缠论 | 无背驰 = 趋势未完 |
| 蜡烛图 | 无顶部形态（黄昏星/乌云盖顶/上吊线） |
| 量价 | 缩量上涨 = 健康 |
| 真技术 | 无逻辑破坏的理由 = 持有 |

### 危险信号
- 缠论：出现背驰
- 蜡烛图：顶部形态出现
- 量价：量能异常放大（巨量）
- 真技术：市场比我们还急 = 见顶信号

---

## 五、止盈标准

### 止盈原则
> 逻辑兑现时止盈，不预测顶

### 止盈触发
| 维度 | 标准 |
|------|------|
| 缠论 | 出现背驰（一卖信号） |
| 蜡烛图 | 顶部形态确认 |
| 量价 | 出现天量或倍量滞涨 |
| 真技术 | 目标位到达 或 逻辑完全兑现 |

### 分批止盈
- 涨到目标位先出一半
- 剩余仓位等背驰信号再出
