# 蜡烛图量价关系

## 基础形态

### 单根K线
- 阳线（收盘>开盘）= 多方占优
- 阴线（收盘<开盘）= 空方占优
- 十字星（收盘≈开盘）= 多空犹豫
- 锤子线（下影线长>2倍实体）= 底部信号
- 射击之星（上影线长>2倍实体）= 顶部信号

### 多根K线组合
- 吞没形态（阳包阴=买入/阴包阳=卖出）
- 孕线形态（十字星+前后K线）
- 早晨之星（底分型）= 买入信号
- 黄昏之星（顶分型）= 卖出信号

## 量价配合（核心）

| 价格形态 | 成交量 | 含义 |
|----------|--------|------|
| 价涨 | 量增 | 多方强劲，上涨确认 |
| 价涨 | 量缩 | 上涨乏力，警惕回调 |
| 价跌 | 量增 | 空方强劲，下跌确认 |
| 价跌 | 量缩 | 下跌减缓，可能见底 |
| 突破放量 | - | 真突破，可信度高 |
| 突破缩量 | - | 假突破，可信度低 |
| 底部放量 | - | 资金介入，可能见底 |
| 顶部放量 | - | 主力出货，警惕见顶 |

## 综合评分加分规则

在缠论+RSI+MACD基础上，增加量价评分：

### 买入信号加分（+15分）
- 底部锤子线 + 量增 = +10
- 早晨之星（底分型）+ 量增 = +10
- 阳包阴 + 量增 = +8
- 底部放量 + 缠论买点 = +5
- 缩量回调 + 缠论买点 = +5

### 卖出信号减分（-15分）
- 顶部射击之星 + 量增 = -10
- 黄昏之星（顶分型）+ 量增 = -10
- 阴包阳 + 量增 = -8
- 顶部放量 + 缠论卖点 = -5

### 中性信号（0分）
- 十字星 = 观望
- 缩量盘整 = 等待方向

## 注意事项
- 量价分析需要结合具体位置（底部/顶部/中途）
- 单一信号不构成决策，需多信号共振
- 缠论结构优先，量价作为辅助确认
