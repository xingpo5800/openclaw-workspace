## 一、四维量化分析体系

### 第1维：缠论（结构）
- K线 → 包含处理 → 分型 → 笔 → 线段 → 中枢 → 背驰 → 买卖点
- 核心：走势终将完美，中枢定义趋势，macd面积判断背驰
- 包含关系：**向上包含取较高值，向下包含取较低值**（之前写反过）

### 第2维：蜡烛图（时机）
- 单根：锤子线（底）、射击之星（顶）、十字星（犹豫）
- 双根：吞没（阳包阴买入/阴包阳卖出）、孕线
- 三根：早晨之星（买入）、黄昏之星（卖出）
- 关键：位置决定信号强度，上涨趋势的底分型比下跌趋势的底分型更可信

### 第3维：量价（能量）
- 平量/倍量/巨量/天量量化标准
- **量价核心规律（2026-03-26实战提炼）：**
  - 放量+上涨 = 真实突破 ✅
  - 缩量+上涨 = 主力控盘，可信 ✅
  - 放量+下跌 = 主力出货 🚨
  - 缩量+下跌 = 抛压轻，可能见底 ✅
- 量比在临界区间（0.85-1.05）时不要急着买，等确认
- 底部放量 = 资金介入；顶部放量 = 主力出货
- 突破必须放量，假突破往往缩量
- 量能斜率 = (近期均量-远期均量)/远期均量，>20%为明显增量
- **5日均量上穿60日均量策略**：slope>30 + RSI<52，成功率70%+，期望值+9%/次
- **临界区间策略（2026-03-26修正）：** slope>20 + RSI<55 + 量比0.70~1.20，更宽松

## 二、三大选股策略

### 策略1：5日量上穿60日均量
- slope > 30（5日均量相对5日前增长）
- RSI < 52
- 量比 0.85~1.05（上穿临界）
- 成功率70%+，期望值+9%/次
- **扫描范围：主板A股（去掉ST股、去掉北交所）≈ 4000只**
- **分批策略：先拉涨幅前500只筛slope>20做候选，再详细算RSI/量比，命中即推**

### 策略2：回踩不破
- 前提：均线多头排列（MA5>MA10>MA20）
- 股价上涨后回踩均线（MA5/MA10）附近
- 回调时缩量（量比<0.6）
- 在均线附近止跌企稳
- K线翻红/底分型确认
- 介入区间：均线价格上下3%

- 资金持续流入，主力建仓信号
- 板块内个股共振机会
