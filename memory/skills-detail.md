# 技能详细说明

## 🌐 网络相关技能（详细）

### web-fetch - 网页内容获取
**功能**: 获取网页内容并转换为markdown或text格式

**参数说明**:
- `url`: 必需，要获取的网页URL
- `extractMode`: 可选，"markdown"或"text"，默认为"markdown"
- `maxChars`: 可选，最大字符数，默认无限制

**使用场景**:
- 获取静态页面内容
- 提取文章正文
- 收集网页信息

**限制**:
- 只能读取内容，无法交互
- 可能被反爬虫机制阻止
- 需要目标网站允许访问

### web-search - 网络搜索
**功能**: 使用Brave Search API进行网络搜索

**参数说明**:
- `query`: 必需，搜索查询字符串
- `count`: 可选，返回结果数量（1-10），默认10
- `country`: 可选，2字母国家代码，默认"US"
- `language`: 可选，ISO 639-1语言代码
- `freshness`: 可选，时间过滤："day"、"week"、"month"、"year"

**使用场景**:
- 信息检索和资料收集
- 最新资讯获取
- 多语言内容搜索

**优势**:
- 支持区域和语言设置
- 返回高质量搜索结果
- 支持时间过滤

### agent-browser - 浏览器自动化（基础）
**功能**: 无头浏览器自动化操作

**核心能力**:
- 页面导航和点击
- 表单填写和提交
- 截图和内容提取
- JavaScript执行

**使用场景**:
- 网页自动化操作
- 数据抓取
- 用户模拟操作

**限制**:
- 需要学习具体API
- 可能被反自动化检测
- 复杂交互可能不稳定

### browser-use - 浏览器自动化（vercel）
**功能**: 高级浏览器交互自动化

**特点**:
- 来自vercel-labs，高可靠性
- 支持复杂交互逻辑
- 更好的错误处理
- 智能等待机制

**使用场景**:
- 复杂的网页操作流程
- 需要智能判断的场景
- 高可靠性的自动化需求

## 📁 文件操作技能（详细）

### read - 文件读取
**功能**: 读取文件内容

**参数说明**:
- `path`/`file_path`: 必需，文件路径
- `offset`: 可选，起始行号（1-indexed）
- `limit`: 可选，最大读取行数

**支持格式**:
- 文本文件
- 图片文件（jpg, png, gif, webp）

**使用场景**:
- 内容查看和分析
- 大文件分段读取
- 图片内容分析

### write - 文件写入
**功能**: 创建和写入文件

**参数说明**:
- `path`/`file_path`: 必需，文件路径
- `content`: 必需，要写入的内容

**特性**:
- 自动创建父目录
- 覆盖已有文件
- 支持文本内容

**使用场景**:
- 创建新文件
- 保存执行结果
- 生成配置文件

### edit - 文件编辑
**功能**: 精确编辑文件内容

**参数说明**:
- `path`/`file_path`: 必需，文件路径
- `oldText`/`old_string`: 必需，要替换的精确文本
- `newText`/`new_string`: 必需，新文本

**要求**:
- oldText必须完全匹配（包括空格和换行）
- 适合精确的文本替换

**使用场景**:
- 修改配置文件
- 更新代码片段
- 精确的内容替换

### exec - 命令执行
**功能**: 执行shell命令

**参数说明**:
- `command`: 必需，要执行的命令
- `workdir`: 可选，工作目录
- `timeout`: 可选，超时时间（秒）
- `background`: 可选，后台执行
- `pty`: 可选，使用伪终端

**特性**:
- 支持环境变量设置
- 支持后台执行
- 支持超时控制
- 支持TTY模式

**使用场景**:
- 系统操作
- 脚本执行
- 程序运行

## 🔍 搜索发现技能（详细）

### find-skills - 技能发现工具
**功能**: 从skills.sh生态系统搜索技能

**核心命令**:
- `npx skills find [query]`: 搜索技能
- `npx skills add <package>`: 安装技能
- `npx skills check`: 检查更新
- `npx skills update`: 更新所有技能

**搜索策略**:
1. 使用具体关键词
2. 优先选择高安装量
3. 选择知名来源（如vercel-labs）
4. 根据功能需求筛选

**安装策略**:
- 使用 `-g -y` 参数简化安装
- 安装失败时多试几次
- 优先选择适合的技能

### clawhub - 技能安装工具
**功能**: 搜索和安装精选技能

**核心命令**:
- `npx clawhub search [keyword]`: 搜索技能
- `npx clawhub install [skill]`: 安装技能
- `npx clawhub list`: 列出已安装技能

**评分系统**:
- 基于用户使用评分
- 0-5分评分制
- 评分越高质量越好

**选择策略**:
- 优先选择高评分技能
- 选择功能匹配度高的
- 避免安装不合适的技能
