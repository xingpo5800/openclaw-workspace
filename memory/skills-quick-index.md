# 快速技能索引

## 🌐 网络相关技能

### 基础网络
- **web-fetch**: 网页内容获取（只读）
  - 功能：获取网页内容并转换为markdown/text
  - 限制：只能读取，无法交互
  - 适用：静态页面内容获取

- **web-search**: 网络搜索
  - 功能：使用Brave Search API进行网络搜索
  - 优势：支持区域和语言设置
  - 适用：信息检索和资料收集

### 浏览器自动化
- **agent-browser**: 浏览器自动化（基础版本）
  - 功能：无头浏览器自动化操作
  - 支持：点击、导航、表单填写等
  - 适用：网页交互和自动化操作

- **browser-use**: 浏览器自动化（vercel版本）
  - 功能：高级浏览器交互自动化
  - 来源：vercel-labs，高可靠性
  - 适用：复杂的网页操作场景

## 📁 文件操作技能

### 基础文件操作
- **read**: 文件读取
  - 功能：读取文件内容（文本和图片）
  - 支持：大文件分段读取
  - 适用：内容查看和分析

- **write**: 文件写入
  - 功能：创建和写入文件
  - 支持：自动创建父目录
  - 适用：文件创建和内容保存

- **edit**: 文件编辑
  - 功能：精确编辑文件内容
  - 要求：oldText必须完全匹配
  - 适用：文件内容修改

### 高级文件管理
- **exec**: 命令执行
  - 功能：执行shell命令
  - 支持：后台执行、超时设置、工作目录
  - 适用：系统操作和脚本执行
