## 🔄 记忆加载时序图

```
启动流程
┌─────────────────────────────────────────────────────────┐
│ 1. 读取SOUL.md (AI个性设定)                              │
│ 2. 读取USER.md (用户信息)                               │
│ 3. 读取IDENTITY.md (身份信息)                           │
│ 4. 读取AGENTS.md (工作区配置)                           │
│ 5. 读取memory/rules-index.md (规则索引)                 │
│ 6. 读取memory/YYYY-MM-DD.md (今日记忆)                  │
│ 7. 根据内存搜索结果读取对应规则                         │
└─────────────────────────────────────────────────────────┘

执行流程
┌─────────────────────────────────────────────────────────┐
│ 1. 用户请求 → 搜索 → 匹配 → 执行 → 反馈 → 更新         │
│ 2. memory_search (智能匹配规则)                          │
│ 3. memory_get (读取匹配规则)                            │
│ 4. 执行具体任务                                          │
│ 5. 更新记忆                                              │
└─────────────────────────────────────────────────────────┘
```

```

    ↓

    ↓

    ↓
┌─────────────────────────────────────────────────────────┐
│ 记忆更新 (新信息存入)                                  │

└─────────────────────────────────────────────────────────┘
```
