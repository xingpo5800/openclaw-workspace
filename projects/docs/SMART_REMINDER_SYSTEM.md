# OpenClaw 智能提醒系统

## 🎯 系统概述

智能提醒系统是OpenClaw的"神经系统"，主动监控和关心系统的各个方面，确保一切正常运行。

## 🚀 核心功能

### 1. 任务管理监控
- **高优先级任务提醒**：发现高优先级待办任务立即提醒
- **任务超期检查**：检查超过3天未处理的任务
- **任务状态跟踪**：实时监控任务执行情况

### 2. 规则体系检查
- **规则索引完整性**：确保规则文件与索引一致
- **规则审查提醒**：定期提醒进行规则审查
- **规则执行验证**：检查规则执行情况

### 3. 备份状态监控
- **备份执行检查**：确认每日备份是否完成
- **GitHub同步状态**：验证远程备份同步情况
- **备份空间监控**：监控备份磁盘使用情况

### 4. 系统健康检查
- **磁盘空间监控**：防止磁盘空间不足
- **内存使用监控**：跟踪系统资源使用情况
- **服务状态检查**：监控关键服务运行状态

### 5. 记忆系统维护
- **记忆文件大小监控**：防止记忆文件过度膨胀
- **记忆完整性检查**：确保记忆系统正常运行
- **记忆优化提醒**：提醒优化记忆内容

## ⏰ 执行时间

### 定时执行
- **每日上午9:00**：执行完整的智能提醒检查
- **备份前30分钟**：提醒准备备份
- **任务截止前**：任务到期提醒（待实现）

### 手动执行
```bash
# 立即执行智能提醒检查
/Users/kevin/.openclaw/workspace/scripts/smart-reminder.sh

# 查看今日提醒
cat /Users/kevin/.openclaw/workspace/logs/alerts-$(date +%Y-%m-%d).txt

# 查看历史提醒
ls /Users/kevin/.openclaw/workspace/logs/alerts-*.txt
```

## 🔧 配置管理

### 启用/禁用提醒
```bash
# 启用智能提醒
launchctl load /Users/kevin/.openclaw/smart-reminder.plist

# 禁用智能提醒
launchctl unload /Users/kevin/.openclaw/smart-reminder.plist

# 查看状态
launchctl list | grep smart-reminder
```

### 自定义检查项目
可以编辑 `/Users/kevin/.openclaw/workspace/scripts/smart-reminder.sh` 添加新的检查项目。

### 提醒阈值调整
在脚本中调整各项检查的阈值：
- 任务超期天数：`-mtime +3`
- 磁盘空间警告：`80%`
- 内存使用警告：可自定义
