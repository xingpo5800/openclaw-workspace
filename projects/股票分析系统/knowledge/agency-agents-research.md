# Agency Agents研究

## 项目信息
- GitHub: msitarzewski/agency-agents
- ⭐ 59,130（最高）
- 180个AI专家agent集合

## 核心设计

每个agent包含：
- **Identity** — 身份和性格特征
- **Core Mission** — 核心任务和工作流
- **Technical Deliverables** — 技术交付物和代码示例
- **Success Metrics** — 成功指标
- **Communication Style** — 沟通风格

## agent类型（部分）

## 对我的价值

### 可以借鉴的设计

1. **agent = 身份 + 专长 + 流程 + 指标**
   - 我现在的agent设计太简单
   - 应该加上性格描述和沟通风格

2. **交付物明确**
   - 每个agent要知道"我要交付什么"
   - 股票研究员：交付物 = 早报、信号、风险评估

3. **成功指标量化**
   - "做得好不好"要有标准
   - 比如：信号准确率、胜率、盈亏比

### 实践应用

我可以为毅哥的工作室设计一套agent模板：

| agent | 角色 | 交付物 |
|------|------|-------|
| 灵犀（我） | 导演/协调者 | 决策、方向把控 |
| 股票研究员 | 市场分析 | 早报、信号、策略 |
| 动画导演 | 内容创作 | 剧本、分镜、素材 |
| 任务执行者 | 具体执行 | 代码、报告、合成 |

## 研究结论
- ⭐最高的项目，设计最成熟
- agent设计的最佳实践：身份+专长+流程+指标
- 我应该为自己的特定任务设计标准化的agent模板
