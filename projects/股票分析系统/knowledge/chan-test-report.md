## 一、实现的三个核心模块

### 1. 中枢识别（Chan.find_zhongshu）
**算法**：三个连续同方向笔的重叠区域构成中枢
- 上升中枢：ZG = max(三个起始价)，ZD = min(三个结束价)
- 下降中枢：ZG = min(三个起始价)，ZD = max(三个结束价)
- 支持相邻中枢自动合并（中枢扩展）

### 2. 背驰判断（Chan.find_beichi）
**双模式检测**：
- 模式A（标准）：离开中枢后连续两笔方向相同，直接比较力度
- 模式B（宽松）：最近两对同方向笔比较力度（允许隔一笔）
**力度指标**：MACD histogram面积
**背驰条件**：价格创新高/低，但力度（MACD面积）减弱

### 3. 三类买卖点（Chan.find_maidian）
- **1买/1卖**：底/顶背驰（最强信号）
- **2买/2卖**：1买后回调不破新低 / 1卖后反弹不创新高
- **3买/3卖**：中枢后回调不进入中枢 / 反弹不进入中枢

### 4. 综合评分（Chan.calc_comprehensive_score）
| 维度 | 最高分 |
|------|--------|
| 缠论买点 | 3分 |
| RSI状态 | 2分 |
| MACD状态 | 2分 |
| 中枢位置 | 2分 |
| 趋势方向 | 1分 |

## 四、观察与结论

1. **缠论信号在A股有效**：两股均识别出底背驰买点，与价格实际走势吻合
2. **背驰检测需要足够历史**：500天数据比120天数据效果显著更好（中枢数3个 vs 1个）
3. **模式B宽松检测有价值**：标准模式未找到信号时，宽松模式补充发现背驰
4. **综合评分实用**：将缠论与RSI/MACD结合，分数直观反映多维度信号
5. **605117当前超买**：RSI=76，建议谨慎（缠论买点已过最佳买入时机）
