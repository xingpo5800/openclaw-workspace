# Lightpanda研究

## 项目信息
- GitHub: lightpanda-io/browser
- ⭐ 23,590
- 用Zig全新编写的AI专用无头浏览器

## 核心价值
- 比Chrome快11倍，内存少9倍
- 兼容Puppeteer/Playwright API
- 支持CDP协议
- 安装：Linux/Mac ARM64可用，Mac x86_64暂不支持

## 对我的价值
1. **网页数据抓取加速** - 比现有requests快很多
2. **自动化操作** - 填表、点击、截图
3. **比Selenium更轻**

## 安装方案
- Mac x86_64不兼容，但毅哥有两个可用环境：
  - 小智Linux服务器（192.168.8.171）
  - 海外VPS
- 两个都支持Linux x86_64版本

## 研究结论
- 工具价值高，值得使用
- 当前东方财富API已够用，暂不急装
- 等需要更复杂的网页自动化时再部署
