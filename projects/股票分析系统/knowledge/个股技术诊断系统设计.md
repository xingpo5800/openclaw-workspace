# 个股技术诊断系统设计方案

> 目标：输入股票代码，输出专业诊断报告  
> 核心理念：多维度综合诊断，看穿大资金痕迹  
> 进度：设计阶段

---

## 一、系统定位

**诊断 vs 预测**

- 诊断：告诉这股票"现在是什么状态"
- 预测：不靠谱，大资金随时可以改变方向
- **核心原则**：诊断是为了找到"大资金的痕迹"，跟着吃肉而不是预判方向

---

## 二、诊断维度（6大维度）

### D1 · 趋势诊断（权重25%）

**判断：当前处于什么趋势**

指标：
- MA5/10/20/60 多头排列/空头排列/缠绕
- 价格与MA的关系
- 趋势强度（ADX）
- 波浪位置（上升5浪？还是下降A浪？）

评分逻辑：
- MA多头排列且价格>MA20>MA60 → 强多头
- 均线缠绕 → 震荡
- MA空头排列 → 趋势下跌

### D2 · 动量诊断（权重20%）

**判断：多空力量谁在主导**

指标：
- RSI（超买超卖）
- MACD（零轴位置/金叉死叉）
- KDJ（超买超卖）

评分逻辑：
- RSI 40-60 → 平衡区域
- RSI <40 → 超卖（可能是买点）
- RSI >70 → 超买（可能是卖点）
- MACD零轴上方 → 多头动能
- MACD零轴下方 → 空头动能

### D3 · 量价诊断（权重25%）——最核心

**判断：大资金在干什么**

指标：
- 今日成交量 vs 5日均量
- 放量上涨？缩量下跌？
- 量能突破前期高点
- 内盘vs外盘（大单方向）

评分逻辑（按毅哥教导）：
- **低位放量** → 大资金进场 ✅
- **高位放量** → 大资金撤退 ⚠️
- **缩量上涨** → 虚假繁荣，小心
- **缩量下跌** → 可能见底

### D4 · 形态诊断（权重15%）

**判断：当前处于什么形态结构**

K线形态：
- 早晨之星/锤子线 → 底部信号
- 黄昏之星/射击之星 → 顶部信号
- 吞没形态 → 趋势反转
- 十字星 → 犹豫信号

支撑阻力：
- 前高/前低
- 整数关口
- 成交密集区

### D5 · 相对强弱（权重10%）

**判断：比大盘强还是弱**

- 与上证/创业板对比
- 与同板块对比
- 相对强度（RS指标）

逻辑：强势股在大盘跌的时候跌得少，大盘涨的时候涨得多

### D6 · 风险诊断（权重5%）

**判断：当前风险有多大**

- RSI极值（>80/<20）
- 偏离MA过大
- 波动率异常
- 质押/解禁风险（基本面）

## 三、综合评分体系

```
总分 = D1×25% + D2×20% + D3×25% + D4×15% + D5×10% + D6×5%

评分区间：
  80-100  →  ⭐⭐⭐ 强烈买入信号
  60-80   →  ⭐⭐   买入信号
  40-60   →  ⭐     观望
  20-40   →  ⚠️    卖出信号
  0-20    →  🚨    强烈卖出信号
```

## 五、技术实现路线

**Phase 1：基础框架（今天完成）**
```
diagnose.py
├── get_data()        — 数据获取
├── calc_indicators() — 计算指标
├── score_dimensions()— 评分
└── generate_report() — 生成报告
```

**Phase 2：深化指标**
```
├── 资金流向（东方财富大单数据）
├── 龙虎榜数据（iflow）
├── 板块轮动分析
└── 相关性分析（与指数/板块对比）
```

**Phase 3：智能诊断**
```
├── K线形态自动识别
├── 缠论中枢/背驰检测
├── 波浪自动计数
└── 买卖点建议
```

**Phase 4：实战集成**
```
├── 接入小智自动盯盘
├── 信号触发自动推送
├── 历史回测验证
└── 策略迭代优化
```

## 七、数据来源

- 实时行情：Tencent API（已验证可用）
- K线数据：东方财富API
- 大单资金：东方财富Level2或iflow
- 板块信息：东方财富板块数据
- 基本面：akshare或akshare本地库

---

## 八、注意事项

1. **不预测，只跟随** — 诊断是现状，不是未来
2. **大资金优先** — 量价信号 > 技术指标
3. **多维度验证** — 单维度信号不作为决策依据
4. **严格止损** — 系统给出信号，不保证100%正确
5. **持续迭代** — 每次实战后复盘，优化参数
