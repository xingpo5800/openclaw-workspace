# 个股智能诊断系统 v2.0 - 网页版

> 定位：专业股票诊断SaaS工具  
> 目标用户：散户投资者、想用AI辅助分析的人  
> 商业模式：卖服务（诊断报告+订阅）  
> 技术栈：Python Flask + HTML/JS + Chart.js  
> 状态：开始构建

---

## 一、核心功能

### 1. 股票搜索
- 输入股票代码或名称
- 自动识别沪深股票
- 搜索历史记录

### 2. 指标选择器（多选）
用户可以自由勾选要分析的指标：

| 分类 | 指标 | 说明 |
|------|------|------|
| 趋势 | MA均线 | 5/10/20/60日均线组合 |
| 趋势 | 布林带(BOLL) | 中轨+上下轨 |
| 趋势 | 道氏理论 | 趋势判断 |
| 趋势 | 波浪理论 | 艾略特波浪计数 |
| 动量 | MACD | 趋势动量 |
| 动量 | KDJ | 随机指标 |
| 动量 | RSI | 相对强弱 |
| 量价 | 成交量 | 量能分析 |
| 量价 | 量价关系 | 放量/缩量信号 |
| 形态 | K线形态 | 早晨之星、黄昏星等 |
| 形态 | 缠论中枢 | 中枢与背驰 |
| 基本面 | 财务指标 | PE、PB、ROE等 |
| 情绪 | 市场情绪 | 资金流向、板块轮动 |

### 3. 诊断报告（核心输出）

**结构：**
```
┌────────────────────────────────────────────┐
│  📊 综合诊断评分 + 操作建议                 │
├────────────────────────────────────────────┤
│  🎯 六大维度评分雷达图                      │
├────────────────────────────────────────────┤
│  📈 智能图表（K线+指标叠加）               │
├────────────────────────────────────────────┤
│  💡 详细分析                               │
│    • 支撑位（多个层级）                    │
│    • 压力位（多个层级）                    │
│    • 进场点建议（含条件）                  │
│    • 止损位                                │
│    • 止盈参考                              │
│    • 风险提示                               │
├────────────────────────────────────────────┤
│  📋 指标解读（用户选了哪些就解读哪些）      │
├────────────────────────────────────────────┤
│  🔮 后市推演（乐观/中性/悲观）             │
└────────────────────────────────────────────┘
```

### 4. 报告解读（关键差异化）
不是只给数字，要给**人话**：
- "RSI=28，超卖区域，历史上有XX次类似情况，反弹概率XX%"
- "价格回踩MA20均线获得支撑，均线从空头转为多头排列"
- "放量突破布林中轨，短期动能转强，第一目标位XX元"

### 5. 历史记录
- 保存诊断过的股票
- 查看历史诊断
- 追踪持仓股票

## 四、诊断引擎设计

### 评分体系（100分制）
```
基础分：50分

+ 趋势维度 (20分)
  MA排列、趋势强度、波浪位置

+ 动量维度 (20分)
  RSI、MACD、KDJ综合判断

+ 量价维度 (20分)
  成交量、量价配合、大单方向

+ 形态维度 (15分)
  K线形态、中枢结构

+ 基本面维度 (15分)
  估值合理性、业绩趋势

+ 情绪维度 (10分)
  资金流向、板块强弱

= 综合评分 0-100
```

### 关键输出计算

**支撑位算法：**
1. 近20日最低点
2. 前期低点（前低）
3. 均线支撑（MA5/10/20/60）
4. 布林下轨
5. 缠论笔低点

**压力位算法：**
1. 近20日最高点
2. 前期高点（前高）
3. 均线压力
4. 布林上轨
5. 整数关口

**进场点算法：**
1. 回踩支撑不破
2. 放量突破压力
3. MACD金叉确认
4. RSI从超卖回升
5. 多指标共振

**止损位算法：**
- 固定止损：成本价 - 3%
- ATR止损：入场价 - 1.5×ATR
- 结构止损：跌破重要支撑

## 六、商业路径

1. **免费试用** → 吸引用户 → 收集反馈
2. **基础版免费，高级指标收费** → 验证付费意愿
3. **定制报告/人工解读** → 高客单价
4. **会员订阅** → 稳定收入

**核心差异化：**
- 不是推荐股票，是教用户"怎么看"
- 给出结论的同时给出"为什么"
- 可视化 + 可解释 = 信任感
