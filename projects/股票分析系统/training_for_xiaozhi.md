### 一、5日均量上穿60日均量策略

**核心逻辑：** 5日均量从下往上接近/上穿60日均量时买入，成功率70%+

**原理（毅哥实战经验）：**
- 5日均量线从下往上接近60日均量线 = 资金开始活跃
- 上穿那一刻 = 主力控盘信号
- 上穿前一天买入 = 成本最低

**量化条件：**
1. 5日均量斜率 > 30（量能明显增加）
2. RSI < 52（还没超买）
3. 20日涨幅 < 30%（位置不高）

**买入时机：**
- 5日均量正在从下往上接近60日均量时买入（不等上穿）

**验证方法：**
- 次日涨幅 > 3% = 主力控盘确认，持有
- 次日小涨 < 2% = 动力不足，止损

### 二、多级别联立判断法

**原理：** 缠论"走势终将完美"——趋势延续有踪迹可循

**用法：**
- 大级别看到不确定信号时，用小级别进一步分析
- 小级别出现底分型 = 调整结束信号
- 小级别出现顶分型 = 反弹结束信号
- 小级别无法创新高 = 可能背驰

**案例（紫金矿业）：**
- 日线：超跌反弹信号（RSI=32.8）
- 30分钟：刚回调完（顶分型）
- 15分钟：已企稳（底分型+up笔）
- 结论：30分钟回调是次级别调整，15分钟企稳说明趋势延续，明天上涨概率大

---

### 三、洗盘识别与二次介入

**洗盘信号：**
- 上穿成功后连续3-5天横盘
- 量能萎缩到极致（量比 < 0.3）
- 回调幅度 < 5%

**二次介入时机：**
- 缩量到极致后再次放量
- 价格突破洗盘区间高点

**核心原则：**
- 不预测主力要洗盘，只跟随信号操作
- 主力洗盘时先出，不在里面等

### 四、完整交易流程

```
1. 发现信号：
   - 5日均量线从下往上接近60日均量
   - RSI < 52
   - 斜率 > 30

2. 买入（上穿前一天）

3. 次日观察：
   - 涨幅 > 3% + 量增 = 主力控盘 ✅ 持有
   - 小涨 < 2% = 动力不足 ⚠️ 观察
   - 大跌 = 失败 ❌ 止损

4. 持有过程中：
   - 等顶部信号（背驰/顶部形态/巨量）
   - 或等洗盘信号

5. 洗盘时：
   - 先出，等二次启动信号
   - 缩量极致 + 放量 = 再进

6. 背驰 = 卖
```
